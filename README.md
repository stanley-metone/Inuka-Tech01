# Week 1 Operations Analyzer

## Overview

This project was completed as part of the Week 1 Operations Analytics assignment.

The project demonstrates how Python can automate operational efficiency tracking across multiple retail stores.

## Repository Contents

- **week1_analyzer.ipynb** – Jupyter Notebook containing the Python solution.
- **Week1_Memo_StanleyMetone.md** – Business memo explaining the solution in non-technical language.

## Features

- Calculates operational efficiency.
- Classifies operational status.
- Generates a formatted operational report.
- Includes beginner-friendly explanations throughout the notebook.
